# EdgeInference

EdgeInference is a gRPC-based server tailored for running ONNX models on edge devices, such as Android, iOS, or other
microcontroller units (MCUs).

- Onnx Runtime

Platform:

- Android, iOS
- Linux, Windows, Mac,
- Raspberry Pi, MCU
